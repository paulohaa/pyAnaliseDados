Antes de começar a brincar...¶

Instalar Python 3¶

Faça o download em https://www.python.org/downloads/ da versão mais recente de Python 3 e instale na sua máquina. Se for um notebook ou micro particular, será tranquilo fazê-lo. Se for corporativo ou compartilhado, você vai precisar de privilégios de administrador na máquina (ou de alguém que o tenha...).

Instalar pacotes Python de captura e análise de dados¶

As instalações de ferramentas em Python são feitas, a priori, por linha de comando. No Windows, por exemplo, utiliza-se o cmd.exe (prompt de comando).

Python dispõe de um instalador padrão para seus pacotes: pip. Ele já vem incorporado às versões mais recentes da linguagem (ou seja, está embutido na instalação anterior). Para instalar pacotes, deve-se utilizar o comando pip install <nomes dos pacotes, separados por espaços simples>. Para este curso, devemos executar o comando abaixo:

pip install numpy pandas matplotlib jupyter

Outros pacotes a incluir: requests beautifulsoup4 twitter wbdata lxml tweepy uber-rides PyPDF2

Em redes corporativas, pode haver problema na execução em função da segurança de rede da instituição. Uma forma de resolver esse problema é citar explicitamente o proxy internet da rede, além do id e senha do usuário de rede. O comando seria conforme abaixo:

pip install <nomes dos pacotes> --proxy http://<id do usuário>:<senha do usuário>@<nome do proxy internet da rede local>

Instalar Jupyter Notebook¶

pip install jupyter

O que é um Jupyter Notebook?¶

Neste caso, o notebook é um formato de documento que contém tanto códigos quanto elementos de texto, figuras, links, vídeos, equações, etc. Como podemos misturar texto e código em um mesmo lugar, esses documentos são lugares ideais para gerar e descrever análises onde tanto a formulação da ideia, a visualização do resultado e a descrição do resultado pode ser feita num mesmo lugar.

Células¶

Há dois tipos principais de células:

  • Markdown: células, como esta, utilizadas para descrever o que está sendo implementado. Elas utilizam alguns caracteres especiais para formatar o texto, como ** texto em negrito ** para fazer texto em negrito;
  • Code: trechos de código para executar os comandos necessários para a análise. No nosso caso, os comandos serão escritos em Python 3, mas o Jupyter Notebook tem suporte para outras linguagens também. Mostraremos alguns comandos simples aqui, mas se quiser saber um pouco mais, segue uma referência às regras básicas da linguagem.

Fórmulas matemáticas nas células de texto (markdown)¶

Também é possível usar fórmulas junto com o texto. Isso é muito útil quando se tem uma explicação matemática necessária para entender o código. A escrita é feita utilizando uma sintaxe específica para inserir fórmulas no texto:

$$c = \sqrt{a^2 + b^2}$$

Células de código (code)¶

Cada célula é um ambiente isolado que pode ser rodado separadamente. Você pode rodar cada célula individualmente e na sequência que desejar, mas as variáveis que ela usa precisam estar referenciadas antes do seu uso.

# Esta é uma célula de código 
for i in range(0, 5):
    print(i)

Além de processar, o código pode ser usado para visualizar dados¶

O código a seguir utiliza o pacote matplotlib para exibir um gráfico simples.

Na área de imports colocamos um %matplotlib inline, certo? Essa é uma das "mágicas" disponíveis no Jupyter, que permite que os gráficos do matplotlib sejam apresentados dentro do próprio notebook, como podemos ver abaixo.

In [2]:
# Esta é uma célula de código

import matplotlib.pyplot as plt
%matplotlib inline 
plt.plot(range(0,100))
Out[2]:
[<matplotlib.lines.Line2D at 0x258de4bcef0>]

E, agora, combinando com numpy para ter uma curva mais elaborada.

Além disso, são adicionados alguns metadados ao gráfico que, além de ser exibido, é salvo no arquivo teste.png.

In [3]:
import numpy as np

# Montando a curva, com duas sequências de números - a segunda é derivada da primeira por uma fórmula
x = np.arange(0.0, 2.0, 0.01)
y = np.sin(2*np.pi*x)
plt.plot(x, y)

# Inserindo metadados e formatação
plt.xlabel('X (raio)')
plt.ylabel('Y (seno da circunferência de raio X)')
plt.title('Uma curva um pouco mais sofisticada')
plt.grid(True)

# Salvando em arquivo
plt.savefig("teste.png", format="png")
