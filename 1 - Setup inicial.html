Antes de começar a brincar...¶

Instalar Python 3¶

Faa o download em https://www.python.org/downloads/ da verso mais recente de Python 3 e instale na sua mquina. Se for um notebook ou micro particular, ser tranquilo faz-lo. Se for corporativo ou compartilhado, voc vai precisar de privilgios de administrador na mquina (ou de algum que o tenha...).

Instalar pacotes Python de captura e análise de dados¶

As instalaes de ferramentas em Python so feitas, a priori, por linha de comando. No Windows, por exemplo, utiliza-se o cmd.exe (prompt de comando).

Python dispe de um instalador padro para seus pacotes: pip. Ele j vem incorporado s verses mais recentes da linguagem (ou seja, est embutido na instalao anterior). Para instalar pacotes, deve-se utilizar o comando pip install <nomes dos pacotes, separados por espaos simples>. Para este curso, devemos executar o comando abaixo:

pip install numpy pandas matplotlib jupyter

Outros pacotes a incluir: requests beautifulsoup4 twitter wbdata lxml tweepy uber-rides PyPDF2

Em redes corporativas, pode haver problema na execuo em funo da segurana de rede da instituio. Uma forma de resolver esse problema citar explicitamente o proxy internet da rede, alm do id e senha do usurio de rede. O comando seria conforme abaixo:

pip install <nomes dos pacotes> --proxy http://<id do usurio>:<senha do usurio>@<nome do proxy internet da rede local>

Instalar Jupyter Notebook¶

pip install jupyter

O que é um Jupyter Notebook?¶

Neste caso, o notebook um formato de documento que contm tanto cdigos quanto elementos de texto, figuras, links, vdeos, equaes, etc. Como podemos misturar texto e cdigo em um mesmo lugar, esses documentos so lugares ideais para gerar e descrever anlises onde tanto a formulao da ideia, a visualizao do resultado e a descrio do resultado pode ser feita num mesmo lugar.

Células¶

H dois tipos principais de clulas:

  • Markdown: clulas, como esta, utilizadas para descrever o que est sendo implementado. Elas utilizam alguns caracteres especiais para formatar o texto, como ** texto em negrito ** para fazer texto em negrito;
  • Code: trechos de cdigo para executar os comandos necessrios para a anlise. No nosso caso, os comandos sero escritos em Python 3, mas o Jupyter Notebook tem suporte para outras linguagens tambm. Mostraremos alguns comandos simples aqui, mas se quiser saber um pouco mais, segue uma referncia s regras bsicas da linguagem.

Fórmulas matemáticas nas células de texto (markdown)¶

Tambm possvel usar frmulas junto com o texto. Isso muito til quando se tem uma explicao matemtica necessria para entender o cdigo. A escrita feita utilizando uma sintaxe especfica para inserir frmulas no texto:

$$c = \sqrt{a^2 + b^2}$$

Alm disso, so adicionados alguns metadados ao grfico que, alm de ser exibido, salvo no arquivo teste.png.

import numpy as np

# Montando a curva, com duas sequncias de nmeros - a segunda  derivada da primeira por uma frmula
x = np.arange(0.0, 2.0, 0.01)
y = np.sin(2*np.pi*x)
plt.plot(x, y)

# Inserindo metadados e formatao
plt.xlabel('X (raio)')
plt.ylabel('Y (seno da circunferncia de raio X)')
plt.title('Uma curva um pouco mais sofisticada')
plt.grid(True)

# Salvando em arquivo
plt.savefig("teste.png", format="png")
